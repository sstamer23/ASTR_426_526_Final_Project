# Yifu's Code Folder

This folder has Yifu's code and data for the project.

The "Cut2_comb_12_non_ss_mag.csv" is the data file from Gaia DR3, the data have more information were pick up by hand.

The "apogee_stars_RA_Dec_cut2.csv" is the data file from APOGEE.

The "Cut2_comb_12_non_ss_mag.ipynb" have two parts, the first part is showing the distribution of RA and DEC, which help us determind the data range for the dataset. The second part is showing the distribution of RUWE, which different colors represent different "non_single_star" values.

The "readCSV.ipynb" is the code for reading the "Cut2_comb_12_non_ss_mag.csv" data and plot the HR diagram.
