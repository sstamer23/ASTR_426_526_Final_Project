# Olwyn's Code Folder
This folder has Olwyn's code and data for the project.
***astr_gaialist.ipynb is likely what you're looking for. This file has all of the queries to GAIA, as well as the queries from individual sample files. This automatically saves the data into a file on your computer. 
***astr_GaiaStatsHist is for statistics, showing standard deviation, an exponential fit, and flagging objects within one standard deviation of the mean. The histograms themselves automatically save themselves, and all inputs that are flagged at a RUWE above ~1.4 are saved in a separate .csv file.
***Note: GAIA async queries are (as of writing this) unstable, and thus using sync queries are better for large datasets. Although, async queries worked for this purpose with a list of ~4 thousand inputs.
